# Feature Ideas

## Praktische Erweiterungen

- [ ] **Manuelle Key-Auswahl für Arrays** - Der Benutzer kann selbst wählen, welcher Key für den Array-Vergleich verwendet wird (statt automatische Erkennung)

- [ ] **Ignore-Liste** - Bestimmte Properties immer ignorieren (z.B. `updatedAt`, `createdAt`, `_id`) - nützlich bei API-Responses

- [ ] **Inline-Diff im JSON-Editor** - Unterschiede direkt im JSON farblich hervorheben (rot/grün wie bei Git)

- [ ] **Merge-Funktion** - Einzelne Werte von A nach B übernehmen (oder umgekehrt) per Klick

- [ ] **URL-Import** - JSON direkt von einer URL laden (z.B. API-Endpoint)

- [ ] **JSON-Path kopieren** - Button um den Pfad eines Unterschieds zu kopieren (`friends[vorname=stefan].hobbies`)

## Nice-to-have

- [ ] **Vergleich speichern/teilen** - Als Link mit Base64-encoded JSON oder LocalStorage

- [ ] **Light/Dark Mode Toggle**
